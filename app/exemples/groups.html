Группы в регулярных выражениях записываются в скобках: (...). Каждая группа получает имя в виде номера (от 0 и далее). Стоит помнить, что само регулярное выражение тоже является группой.

Группы нумеруются прямым рекурсивным левосторонним обходом в глубину. Простой способ определять номер группы - считать открывающиеся скобки, представляя что все выражение тоже обернуто в скобки.

Например, в выражении /a(b(c)(d))/ будут группы:
0 = само выражение = a(b(c)(d)), 1 = b(c)(d), 2 = c, 3 = d.

Группы используются для поиска в тексте при сопоставлении и для сложных замен.
